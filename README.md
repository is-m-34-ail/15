# Todo App на React

Приложение для управления задачами с функциями:
- ✅ Добавление задач
- ✏️ Редактирование задач  
- 🗑️ Удаление задач
- 🔄 Фильтрация (Все/Активные/Завершённые)
- 📊 Статистика выполнения

## Технологии
- React 18
- CSS3
- Babel
